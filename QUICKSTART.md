# CaseTrack - Quick Start Guide

Welcome to CaseTrack! This guide will help you get started with the advocate case management system.

## Getting Started (2 minutes)

### First Time Login
1. **Register**: Click "Create Account"
2. **Enter Email**: Use any email (e.g., advocate@example.com)
3. **Enter Password**: Create a secure password (min 6 characters)
4. **Confirm Password**: Re-enter your password
5. **Submit**: Click "Create Account"

### First Login
1. The app will load sample cases automatically
2. Explore the dashboard and different views
3. See how cases are organized

## Create Your First Case

### Step 1: Start New Case
- Click the **"+ New Case"** button in the top navigation

### Step 2: Fill Case Details
- **Client Name**: Enter client's full name
- **Case Number**: Enter court case number (e.g., 2024-CV-001)
- **Court Name**: Enter court name (e.g., District Court)
- **Status**: Select "Active" or "Closed"
- **Case Notes**: Add any important information

### Step 3: Add Hearing Dates
- Click **"+ Add Date"** to add first hearing
- **Date**: Select the hearing date
- **Time**: Enter time (optional)
- Click **"Add"** to confirm
- Repeat to add more hearings

### Step 4: Upload Case Files (NEW!)
- Scroll to **"Case Documents & Photos"**
- **Method 1**: Drag and drop files onto the upload area
- **Method 2**: Click "browse files" to select from computer
- Files appear in the list immediately
- Maximum 5MB per file

### Step 5: Create Case
- Click **"Create Case"** button to save everything

## View Your Cases

### Available Views (Click tabs to switch)

1. **All Cases**
   - All your cases sorted by next hearing date
   - Filter by status (All/Active/Closed)
   - Search by client, case number, or court

2. **Today's Hearings**
   - Cases with hearings happening today
   - Quick focus on immediate priorities

3. **Next 7 Days**
   - All hearings scheduled for next week
   - Plan ahead view

4. **Calendar**
   - Monthly calendar showing all hearing dates
   - Visual planning tool
   - Navigate between months

5. **Timeline**
   - Chronological list of all hearings
   - Detailed view of upcoming hearings
   - Shows case and client info for each hearing

## Manage Cases

### View Case Details
1. Click on any case card
2. See all hearings for that case
3. View case notes and files
4. Edit or delete information

### Add More Hearings
1. Open case details
2. Scroll to "Hearing Dates"
3. Fill in date and time
4. Click "Add" button

### Add Case Notes
1. Open case details
2. Click on "Case Notes" section
3. Click to edit
4. Add your notes
5. Click "Save"

### Add Hearing Notes
1. Open case details
2. Click "Edit Notes" on any hearing
3. Add hearing-specific information
4. Changes save automatically

### Upload Files to Existing Case
1. Open case details
2. Scroll to "Case Files & Documents"
3. Drag and drop OR click to browse
4. Files appear immediately

### View/Download Files
- **View Image**: Click image icon to see full size
- **Download File**: Click file icon to download
- **Delete File**: Click X icon to remove

### Delete Case
1. Open case details
2. Click "Delete Case" button
3. Confirm deletion

## Search & Filter

### Quick Search
- Use the search bar at top of cases
- Search by:
  - Client name
  - Case number
  - Court name
- Results update as you type

### Filter by Status
- In "All Cases" view, use dropdown
- "All Status": Show all cases
- "Active": Show only active cases
- "Closed": Show only closed cases

## Settings & Logout

### Change Account
1. Click **Logout** button in header
2. You'll return to login page
3. Enter different email to switch user

### Clear Data
- Each user's data is separate
- Data persists until you clear browser localStorage
- Logging out doesn't delete data

## File Storage Info

### What's Stored Locally
- Your user email
- All cases and hearing dates
- Case notes and files
- User preferences

### Storage Limit
- Varies by browser (typically 5-10MB)
- Files take ~33% more space when encoded
- Regular backups recommended for important files

### Privacy
- All data stored in YOUR browser only
- No data sent to servers
- Complete privacy and control

## Troubleshooting

### Can't Create Case?
- Check all required fields are filled
- Ensure at least one hearing date is added

### File Won't Upload?
- Check file size (max 5MB)
- Ensure file type is supported
- Check browser storage space

### Can't Find Case?
- Use search function
- Check status filter
- Try different search terms

### Cases Disappeared?
- Check you're logged in as correct user
- Clear browser cache (careful!)
- Data persists across sessions
